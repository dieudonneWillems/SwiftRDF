  • ==(_:_:)

    Compares two blank nodes and returns true when both have the same identifier. Keep in mind that the identifier of a blank node is only valid within its scope. Blank nodes that come from different scopes should not be compared.

    Declaration

    Swift

    public func == (left: BlankNode, right: BlankNode) -> Bool

    Parameters

    left

    The first blank node used in the comparisson.
    right

    The second blank node used in the comparisson.

    Return Value

    True when the identifier of both blank nodes are equal, or false otherwise.
