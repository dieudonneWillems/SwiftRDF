Vocabulary

public class Vocabulary

This class defines a vocabulary, i.e. a set of URS’s representing concepts in a vocabulary. Subclasses of this class contain sets of static variables.

    Creates a new URI to be included in the vocabulary. This method is a convenience method to create URIs without the need to catch URI initialisation errors. It is imperative that the URIs created are valid URIs, otherwise a runtime error will occur.
