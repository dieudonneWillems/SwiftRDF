  • Decimal

    This type defines a decimal numerical value as defined in the XML Schema namespace. A decimal represents a subset of the real numbers, which can be represented by decimal numerals. The value space of decimal is the set of numbers that can be obtained by multiplying an integer (i.e. decimalInteger) by a non-positive power of ten (i.e. decimalExponent), i.e., expressible as i × 10^-n where i and n are integers and n >= 0. This implementation supports decimals with 19 decimal digits.

  • Duration

    A duration represents a duration of time and is expressed in a combination of six components, the number of years, months, days, hours, minutes, and seconds. There is no determinate relation between the different components, for instance a month may contain 28,29,30, or 31 days and a minute may contain 59, 60, or 61 seconds (due to leap seconds). Durations can only be compared to each other when viewed in their context, i.e. the starting or ending date of the duration when the length of a month is known and when the addition of leap seconds is known.
